
### short cut

```
npm run short-cut
```

or

```
node bin/program.js -p -P -b -c tim
```

```
program.peppers => true
program.pineapple => true
program.bbqSauce => true
program.cheese => 'tim'
```

### full option

```
npm run full-option
```

or

```
node bin/program.js --peppers --pineapple --bbq-sauce --cheese tim
```

```
program.peppers => true
program.pineapple => true
program.bbqSauce => true
program.cheese => 'tim'
```
